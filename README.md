# common_reset
Common used css reset rules, works better with [Normalize.css](https://github.com/necolas/normalize.css).

## NPM
```javascript
npm i common_reset --s // Subject based
```

## Usage
`` import 'common_reset' ``
### Nuxt.js
nuxt.config.js
```javascript
  /*
  ** Global CSS
  */
  css: [
    'common_reset', // Implement ahead of normalize.css is recommended
    'normalize.css'
  ]
```
